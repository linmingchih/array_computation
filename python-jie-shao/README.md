### SpyderIDE(整合開發環境)

提供以下功能

* 編輯器： 具有函式和類檢視器 代碼分析特性 代碼補全 直接跳入定義&#x20;
* 互動視窗： Python或IPython埠都在工作區可以調整和使用。支援對編輯器里的代碼直接除錯。
* 文件瀏覽器： 在編輯器或埠中顯示任意類或函式呼叫的文件。&#x20;
* 變數瀏覽視窗&#x20;
* Matplotlib的圖表顯示視窗&#x20;
* 歷史記錄
